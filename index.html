工作：我如何幫助企業用戶 -- 滿意客戶實例

  • 這是一家位於台中的生產機製造商, 我設計的軟體幫助該公司在去年的台灣塑膠博覽會上展出, 並為該公司贏得多筆海外訂單
  • 我的軟體幫助公司內的人員更快，更好的完成工作, 以下是受惠於我的設計的人
老闆 - 期待能夠一目了然地了解他的生意經營狀況.

過去狀況 : 老闆花很多時間開會，但是無法掌握及時公司狀態。

我的解方 : 提供一個簡潔的儀表板, 上面只顯示老闆所關心的關鍵業務數據。

事實結果 : 老闆把時間花在他喜歡做的事情：與潛在的客戶會面, 帶來更多業務。

使用技術 : Linux, Apache, MySQL Database, PHP, HTML5, CSS3, jQuery, javascript


工程師 - 希望能夠及時監控機器運作，在發生錯誤時在線上進行故障排除，幫助工廠的操作員解決問題.

過去狀況 : 機器壞了，客戶要求維修，機器製造商派工程師到海外工廠修理現場問題。 由於機器在維修期間無法繼續生產，因此至少需要3天的生產停機時間。

我的解方 : 提供帶有及時圖表，警報，歷史表圖,和機器維護記錄的及時監控系統.

事實結果 :

  • 工程師不再需要輪夜班監控機器：系統24小時自動記錄所有參數，包括那些可以幫助工程師找出問題所在的關鍵數據。

  • 預測故障的的能力：當累積了足夠的數據後, 公司可以預測零件何時出現故障，並定時更換零件，防止整台機器停止運作的嚴重性故障。

  • 為公司帶來一門新生意--定期保養：在過去，公司只能在機器壞掉時維修收錢，現在定期保養成為公司穩定的收入來源。

使用技術 : Node.js, MQTT, Websockets, Charts, MSSQL, MySQL, GCM, IOT, Telemetry, Visualization, Anomaly Detection.


工廠客戶 - 希望能夠 輕鬆訂購零件, 保護資產：生產機台, 防止機器停止運作

過去狀況 : 當客戶需要零件時，他們需要打個電話給客服,客服再根據客戶描述找到零件,這很容易雞同鴨講,浪費時間。        

我的解方 : 用帶有QR碼掃描的Android APP訂購零件。 這種方式可以完全消除人為的鍵入錯誤。比線上購物更簡單.

事實結果 :

  • 客戶現在可以輕鬆訂購零件就像購物一樣簡單： 1.掃描QR碼，2.輸入數量 3.按下按鈕訂購。

  • 保護最重要的資產： 生產機台. 一部生產機的造價遠遠超過10輛車，對於工廠而言是一筆不小的投資,透過"定期保養"可以延長機器的壽命.

  • 防止機器停止運作： "定期保養" 可以提早發現故障原因,在零件發生故障之前進行更換零件,防止生產計劃延遲，並按時將貨物交付給客戶

使用技術 : Linux, PHP, Apache, MySQL, HTML5, CSS3, javascript, jQuery


銷售人員 - 希望使用Web平台和APP平台處理零件訂單,來提高工作效率。

過去狀況 : 通過電話接受客戶訂單，僅依賴紙本文件來追踪訂單。

我的解方 : 使用Web平台(辦公室)和APP(在外)追踪訂單。當訂單狀態改變:例如收到訂單,裝運,到貨等,系統會發通知通給手機。

事實結果 : 銷售人員可以在任何地方處理客戶訂單，包括辦公室，出差在外，甚至在高鐵上/飛機上, 都可以處理訂單。

使用技術 : Linux, PHP, Apache, MySQL server, Google Cloud Message (push notification)
